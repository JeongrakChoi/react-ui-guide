# react-ui-guide

React 환경 정적인 UI 페이지 활용 및 개인 공부용

## 터미널

```
최초 실행시
> npm install
> npm start

빌드
> npm run build

gh-pages 배포
> npm run deploy
```

## 메모

- 깃허브 페이지에 호스팅 하기 위한 `gh-pages` 라이브러리 추가
- 404.html 임의 추가 후, index.html로 리다이렉션하는 코드 추가
  깃허브 페이지에서 spa 지원하지 않기 때문  
  (https://iamsjy17.github.io/react/2018/11/04/githubpage-SPA.html)
